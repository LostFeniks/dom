# Whack a Goblin! 🔨

[![Build and Deploy](https://github.com/LostFeniks/dom/actions/workflows/deploy.yml/badge.svg)](https://github.com/LostFeniks/dom/actions/workflows/deploy.yml)

Игра, где гном случайно перемещается по полю 4x4 каждую секунду.

[Посмотреть демо](https://LostFeniks.github.io/dom/)

## 🎮 Описание

Гном (goblin) появляется в случайной ячейке игрового поля 4x4 и каждую секунду перемещается в другую случайную ячейку, никогда не оставаясь на том же месте.

## 🛠 Технологии

- JavaScript (ES6+)
- Webpack (сборка)
- Babel (транспиляция)
- ESLint (линтер)
- Jest (тестирование)
- GitHub Actions (CI/CD)
